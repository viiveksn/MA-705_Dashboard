# MA705 Final Project

This repository contains files used in the MA705 dashboard project.


The final dashboard is deployed on Heroku [here](https://ma705bostonuniversities.herokuapp.com).

## Dashboard Description

Flipkart was the first entrant in India's e-commerce space and in the years following 2010 became a $6-Billion firm,
Mobile devises accounted for 6% of its sales , largest by Product type. 
The Dashboard aims to generate valuable insights on Flipkart mobile sales and its relation with brands and device charactersistics. 

Brief description of dashboard's purpose.

The user will be able to use the insights of the dashboard to check which brands/device features were preferred by customers, what were
the overall sales for the brand.

### Data Sources

Brief description of where/how you got the data and how it was processed.

The data used for the dashboard was sourced from kaggle following extensive research. 
Data was checked for null values and unwanted colums were omited while display the interactive table.
